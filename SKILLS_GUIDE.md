# OpenCode Skills Guide

This project is now configured with the following specialized skills. You can invoke them using the `skill` tool.

## 1. Playwright (Browser Automation)
- **Purpose**: Full browser control for web scraping, automated testing, and UI verification.
- **Key Tools**: `browser_navigate`, `browser_click`, `browser_snapshot`.
- **Usage**: When you need to interact with any website or debug a running frontend app.

## 2. Frontend Engineer (UI/UX)
- **Purpose**: Crafts stunning UI components and layouts.
- **Expertise**: Tailwind CSS, React, Framer Motion, Responsive Design.
- **Usage**: Use for any visual changes or "pixel-perfect" requirements.

## 3. OpenSpec (Architecture & Planning)
- **Purpose**: Manages formal change proposals and technical specifications.
- **Workflow**: `proposal` -> `validate` -> `apply`.
- **Usage**: When planning significant refactors or new features.

## 4. Librarian (Research & Docs)
- **Purpose**: Deep-dives into external library documentation and GitHub repositories.
- **Usage**: When working with unfamiliar APIs or searching for "best practice" examples in Open Source.

## 5. Oracle (Technical Advisor)
- **Purpose**: High-level architectural reasoning and complex debugging.
- **Usage**: When you hit a "wall" or need a second opinion on a critical design decision.

## 6. UI/UX Pro Max
- **Purpose**: Expert-level frontend engineering for premium visual quality.
- **Expertise**: Advanced Framer Motion, A11y, Design Systems, Mobile-First.
- **Usage**: When you need to build high-end components or "wow" the user with aesthetics.
