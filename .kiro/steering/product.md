# AI Life Assistant Product Overview

## Product Description

AI Life Assistant is a comprehensive personal productivity system featuring 5 specialized AI secretaries that provide daily life management support. The system is designed for a specific user (大洪) and includes both a Python CLI application and a modern web application.

## Core Features

### 5 AI Secretaries
- **News Secretary** (新闻秘书) - AI/tech news aggregation and summarization
- **Work Secretary** (工作秘书) - Task management and work planning
- **Outfit Secretary** (穿搭秘书) - Weather-based clothing recommendations
- **Life Secretary** (生活秘书) - Health, diet, and lifestyle management
- **Review Secretary** (复盘秘书) - Evening reflection and personal growth analysis

### Key Capabilities
- Daily routine automation (morning and evening workflows)
- Weather API integration for personalized recommendations
- File-based data persistence with date-organized logs
- Interactive and automated modes
- Web interface for modern user experience

## Target User
- Primary user: 大洪 (37-year-old tech expert in Shanghai)
- Highly personalized system with user-specific preferences
- Focus on productivity, health, and personal growth

## System Architecture
- **Backend**: Python CLI + FastAPI web server
- **Frontend**: Next.js web application
- **Data**: File-based storage with daily logs organized by date
- **AI**: Claude API (Anthropic) or GLM API integration
