# 前端页面优化实施计划

> 创建时间：2026-03-07
> 目标：优化 5 个前端页面的排版、布局、配色和用户体验

---

## 📋 项目概述

**优化目标**：将前端页面打造为个人门户，方便快速浏览

**优先级**：用户体验和 UI 设计 > 性能优化 > 代码重构

**设计风格**：Genesis Design System（极客赛博/未来主义简约）

**参考网站**：https://ai-bot.cn/ （Agents/Tools 页面参考）

---

## 🎯 任务列表

### Task 1: Home 页面美化

**当前状态**：基础布局完成，但间距过大、内容单一

**优化要求**：
- 优化 section 间距（space-y-40 → space-y-24/32）
- 增强 Hero 视觉冲击力（更大的标题、更好的渐变）
- 优化卡片排版和层次
- 添加更多视觉动效
- 保持现有 Genesis Design System 风格

**文件位置**：
- `frontend/src/app/(public)/page.tsx`
- `frontend/src/components/features/home/Hero.tsx`
- `frontend/src/components/features/home/*.tsx`

---

### Task 2: Agents 页面改造（参考 ai-bot.cn）

**当前状态**：3 列网格，卡片设计过大

**优化要求**：
- 改为小长方形卡片（16:9 或 3:2 比例）
- 4 列网格布局（桌面端）
- 紧凑的卡片内容（图标 + 名称 + 短描述）
- 简洁的状态标识
- 参考 ai-bot.cn 的小长方形卡片式简洁网格式布局

**文件位置**：
- `frontend/src/app/(public)/agents/page.tsx`

**设计规范**：
- 卡片比例：16:9
- 桌面网格：4 列
- 平板网格：2 列
- 移动端：1 列

---

### Task 3: Tools 页面改造（参考 ai-bot.cn）

**当前状态**：2+4 布局，旗舰工具卡片过大

**优化要求**：
- 统一为小长方形卡片
- 4-5 列网格布局
- 移除旗舰工具大卡片
- 简洁的图标 + 名称 + 描述
- 参考 ai-bot.cn 的小长方形卡片式简洁网格式布局

**文件位置**：
- `frontend/src/app/(public)/tools/page.tsx`

**设计规范**：
- 卡片比例：3:2 或正方形
- 桌面网格：4-5 列
- 平板网格：2 列
- 移动端：1 列

## 🔧 技术要求

### 通用要求

1. **响应式设计**：移动端自动转换为单栏堆叠
2. **Genesis Design System 配色**：
   ```
   主背景：#0a0a0f (Abyss Black)
   卡片：rgba(26, 26, 36, 0.7) + backdrop-blur
   主色：#00f2ff (电光青)
   强调：#bc13fe (霓虹紫)
   ```
3. **Framer Motion 动画**：所有卡片添加悬停效果和入场动画
4. **可访问性**：确保键盘导航和屏幕阅读器支持

### 代码质量要求

- TypeScript 类型完整
- 无 ESLint 警告
- 组件拆分合理（>200 行需拆分）
- 添加必要的注释
