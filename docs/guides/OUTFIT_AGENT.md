# 穿搭秘书 (Outfit Secretary) 使用指南 👔

## 1. 前言：体面每一天

`OutfitAgent` 旨在消除你清晨出门前的“决策疲劳”。它通过融合实时天气数据与个人审美偏好，为你提供科学、得体且舒适的穿着建议。

---

## 2. 核心功能

- **🌤️ 天气数据集成**：深度对接 QWeather API，实时获取体感温度、风速、降水概率及紫外线强度。
- **🎭 场景化智能适配**：根据当日日程（如会议、约会、运动）自动切换正式或休闲模式。
- **🧤 多维度建议体系**：涵盖主穿搭、替换方案、配饰（手表、领带）及必要的出行提醒（如带伞、防晒）。
- **⚠️ 气温剧变预警**：当检测到温差超过 8℃ 时，自动触发保暖提醒。

---

## 3. 操作步骤

### 🟢 场景一：日常快速决策 (自动运行)
作为晨间工作流的一部分，获取基于默认偏好的建议。

1.  **运行命令**：
    ```bash
    python -m src.cli.main --step outfit
    ```

### 🔵 场景二：特殊场合定制 (交互模式)
适用于今天有重要会议、面试或户外活动等非日常情况。

1.  **启动交互**：
    ```bash
    cd backend
    python -m src.agents.outfit_agent
    ```
2.  **响应询问**：
    -   **特殊场合**：输入 `meeting` 或 `sporty`。
    -   **颜色偏好**：例如 `black and white`。
    -   **舒适度优先级**：输入 1-5 之间的数字。

## 4. 穿搭逻辑说明

| 模式 | 核心关键词 | 典型单品 |
| :--- | :--- | :--- |
| **正式 (Formal)** | 商务、干练 | 西装、皮鞋、修身衬衫 |
| **休闲 (Default)** | 舒适、透气 | Polo 衫、卡其裤、休闲鞋 |
| **运动 (Sporty)** | 灵活、排汗 | 卫衣、慢跑裤、运动鞋 |

---

## 5. 常见问题 (FAQ)

- **API 无法连接？** 系统会自动降级使用“默认上海气温 (22℃)”模式，确保基础服务始终可用。
- **如何修改我的偏好？** 目前可在 `src/agents/outfit_agent.py` 的 `user_preferences` 字典中直接修改颜色避讳和气候偏好。
