# 穿搭秘书 (Outfit Secretary) 使用指南 👔

## 1. 前言：消除决策疲劳

`OutfitAgent` 通过融合实时天气与个人审美，为你提供科学、得体且舒适的穿着建议。

---

## 2. 核心功能

- **🌤️ 天气数据集成**：深度对接 QWeather API，获取体感温度、风速及降水概率。
- **🎭 场景化适配**：支持从 `/chat` 空间告知秘书“今日有重要会议”，自动切换正式模式。
- **💾 偏好记忆**：结合 `VectorMemory` 记录你对特定颜色、材质的喜好。

---

## 3. 操作步骤

### 🟢 场景一：自动流水线
作为 Morning Flow 的一部分运行，建议会自动推送到 Dashboard。

### 🔵 场景二：对话调整
在右下角悬浮 AI 入口或 `/chat` 页面，切换至“穿搭秘书”，输入：
> “今天要出去外拍，帮我选一套活动方便的衣服。”

---

## 4. 故障降级逻辑
如果 API 无法连接：
- 系统会自动降级使用“默认上海气温模式 (18-22℃)”。
- 数据库会记录该状态，并在界面提示“使用离线天气建议”。
