# 复盘秘书 (Review Secretary) 使用指南 🌙

## 1. 前言：连接今日与未来

`ReviewAgent` 是你每日工作流的终点，也是进化的起点。它不仅仅是列出流水账，更是通过深度学习你的行为模式，将“经历”转化为“经验”的智慧中枢。

---

## 2. 核心功能

- **📂 全日志深度聚合**：自动扫描当日所有秘书（新闻、工作、穿搭、生活）的日志，构建全景视野。
- **🧠 结构化反思框架**：基于认知科学，引导用户从成就、障碍、进度、策略四个维度进行思考。
- **💾 长期偏好提取 (v2.0)**：自动识别对话中的倾向性（如“我不喜欢...”、“我希望...”），并更新用户画像。
- **📈 情感与压力监测**：通过语义分析监测当日的心情波动，预判可能的疲劳期。

---

## 3. 操作步骤

### 🟢 场景一：睡前深度对话 (交互模式)
这是最能触发“偏好提取”功能的方式，建议在睡前 30 分钟进行。

1.  **启动脚本**：
    ```bash
    python -m src.cli.main --step review --interactive
    ```
2.  **引导式反思**：根据 AI 的提问（如：“今天最有成就感的一件事是什么？”），输入你的真实感受。

### 🔵 场景二：快速静默复盘 (自动模式)
适用于当天已经记录了足够的任务状态，仅需生成一份书面总结的情况。

1.  **执行命令**：
    ```bash
    python -m src.cli.main --step review
    ```

---

## 4. 复盘思维框架 (S.O.P.)

生成的 `review.md` 遵循以下逻辑：

| 板块 | 关注点 | 价值 |
| :--- | :--- | :--- |
| **🌟 Highlights** | 高光时刻 | 强化正向动机。 |
| **🚧 Obstacles** | 核心障碍 | 识别并规避重复错误。 |
| **📈 Progress** | 长期契合度 | 确保今日工作服务于 v2.0 大目标。 |
| **💡 Strategy** | 明日策略 | 预设明早的“战斗模式”。 |

---

## 5. 记忆演进逻辑

这是 v2.0 的黑科技。当你在复盘中提到：
> “今天下雨去健身房太麻烦了，以后雨天还是在家练。”

`PreferenceExtractor` 会自动捕获并存入 SQLite。下次雨天，生活秘书将**不再**推荐健身房行程，而是直接给出室内拉伸方案。
