# 复盘秘书 (Review Secretary) 使用指南 🌙

## 1. 前言：将经历转化为经验

`ReviewAgent` 是你每日工作流的终点。在 v2.0 中，它负责从分散的数据库记录中提取价值，并更新你的“长期数字镜像”。

---

## 2. 核心功能

- **📂 数据全量聚合**：不再扫描 md 文件，而是直接查询 `news_articles`, `work_tasks`, `health_metrics` 表中的今日增量。
- **🧠 偏好自动提取**：通过对话识别倾向性，并更新 `semantic_memories` 向量库。
- **📈 情感/压力监测**：通过语义分析生成每日心情指数（Mood Summary）。

---

## 3. 操作步骤

### 🟢 场景一：晚间深度复盘 (Web/Chat)
1. 进入 `/chat` 页面，选择“复盘秘书”。
2. 输入今日的感想，或直接点击“生成今日复盘”。
3. 结果将结构化保存至 `daily_reflections` 表。

---

## 4. 复盘维度

| 维度 | 数据来源 | 价值 |
| :--- | :--- | :--- |
| **Productivity** | `work_tasks` 完成率 | 评估工作效能 |
| **Growth** | 对话中的反思文本 | 积累知识资产 |
| **Well-being** | `health_metrics` | 预警健康风险 |

---

## 5. 记忆演进 (Memory Core)
复盘秘书是系统唯一的“写入权”代理，它提取的信息将直接影响次日其他 4 个秘书的初始提示词。
