# 生活秘书 (Life Secretary) 使用指南

生活秘书 (`LifeAgent`) 关注你的身体健康和生活质量，提供饮食、运动和作息的全方位建议。

## ✨ 功能介绍

*   **健康指标追踪**：记录睡眠质量、精力水平、饮水量等。
*   **闭环反馈**：读取昨日日志，根据你昨天的运动和睡眠情况调整今日建议。
*   **个性化饮食**：根据你的口味偏好和健康目标推荐三餐。
*   **作息优化**：提供针对程序员/脑力工作者的作息建议（如护眼、久坐提醒）。

## 💻 使用方式

### 直接运行脚本 (交互模式)

这是最常用的方式，用于晨间打卡：

```bash
cd backend
python -m src.agents.life_agent
```

交互流程：
1.  **昨晚睡眠质量** (excellent/good/fair/poor)
2.  **当前精力水平** (high/medium/low)
3.  **今日是否已运动** (yes/no)
4.  **当前饮水量**

### 通过 CLI 集成运行

```bash
python -m src.cli.main --step life
```

## 📊 数据追踪与反馈

生活秘书不仅是生成建议，还在构建你的健康数据闭环：

1.  **读取**：启动时，它会尝试读取 `data/daily_logs/YESTERDAY/life.md`。
2.  **分析**：提取昨日是否完成了运动目标、睡眠时长是否达标。
3.  **调整**：
    *   如果昨天没睡好 -> 今天建议午休，减少高强度运动。
    *   如果昨天大餐 -> 今天建议清淡饮食。

## 🥗 建议内容

输出报告包含以下板块：

*   **饮食计划**：早餐、午餐、晚餐及下午茶的具体食谱建议。
*   **运动安排**：根据你的精力水平推荐（如：30分钟快走 vs 20分钟HIIT）。
*   **作息建议**：起床、工作间隙休息、睡前准备。
*   **健康小贴士**：针对当下的实用建议。

## ⚙️ 用户画像配置

目前基于硬编码的用户画像（代码位置 `src/agents/life_agent.py`）：

*   **职业**: 技术专家 (意味着久坐、用眼过度)
*   **目标**: 维持体重、每周运动3次
*   **偏好**: 避免过度辛辣油腻，偏好均衡饮食
