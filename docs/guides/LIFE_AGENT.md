# 生活秘书 (Life Secretary) 使用指南 🥗

## 1. 前言：健康即生产力

对于开发者而言，由于长期处于久坐、高强度脑力劳动的状态，身体健康往往容易被忽视。`LifeAgent` 致力于通过结构化的数据追踪与智能建议，帮助大洪维持最佳的身体状态。

---

## 2. 核心功能

- **📊 健康指标追踪**：系统化记录睡眠质量、精力水平及每日饮水量。
- **🔄 闭环反馈机制**：读取昨日日志，根据历史表现（如运动量、睡眠不足）动态调整今日建议。
- **🍽️ 个性化饮食方案**：根据预设的“健康轻食”偏好推荐每日三餐。
- **⏰ 作息深度优化**：针对程序员职业病，提供护眼及久坐提醒。

---

## 3. 使用场景与操作步骤

### 🟢 场景一：晨间快速打卡 (交互模式)
这是最推荐的使用方式，通过简单的问答完成身体状态的同步。

1.  **启动脚本**：
    ```bash
    cd backend
    python -m src.agents.life_agent
    ```
2.  **完成对话**：
    -   选择昨晚睡眠质量 (`excellent` / `good` / `fair` / `poor`)。
    -   输入当前精力等级 (`high` / `medium` / `low`)。
    -   确认今日是否已有运动计划。

### 🔵 场景二：全自动计划生成 (CLI 集成)
作为早间工作流的一部分，无需人工干预即可生成报告。

1.  **运行命令**：
    ```bash
    # 统一通过 CLI 入口运行
    ./scripts/start-dev.sh # 确保服务运行
    python -m src.cli.main --step life
    ```

---

## 4. 建议报告结构

生成的 `life.md` 报告将包含以下关键板块：

| 版块 | 内容描述 |
| :--- | :--- |
| **饮食计划** | 推荐三餐食谱（如：全麦面包+煎蛋）。 |
| **运动安排** | 根据精力推荐强度（如：瑜伽 vs HIIT）。 |
| **健康小贴士** | 实用的微调建议（如：睡前一小时远离蓝光）。 |
| **特别提醒** | 针对连续未达标指标的警告（如：补水预警）。 |
