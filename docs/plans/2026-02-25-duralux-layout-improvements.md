# Duralux 布局改进计划

## 问题分析

### 1. 左侧导航栏 (Sidebar)
**当前问题：**
- 折叠按钮样式不够突出（参考网站：左侧边缘居中，实心圆形）
- 菜单项 hover 效果缺少右侧箭头指示
- 展开态 active 缺少圆角背景包裹
- 缺少用户信息卡片（底部）

**参考网站特征：**
- 折叠按钮在左侧边缘中间位置
- 菜单项 hover 时右侧有小箭头
- active 项有完整的圆角矩形背景
- 底部有用户头像和名称卡片

### 2. 顶部状态栏 (Header)
**当前问题：**
- 搜索框缺少圆角边框和完整样式
- 通知按钮缺少下拉菜单
- 用户头像下拉菜单样式需要优化

**参考网站特征：**
- 搜索框：圆角矩形，键盘图标 + 占位符 + ⌘K 提示
- 通知：带红点的铃铛图标，下拉消息列表
- 主题切换：太阳/月亮图标切换
- 用户：圆形头像 + 下拉菜单（个人资料/设置/登出）

### 3. 图表交互性
**当前问题：**
- StatCard 使用静态 SVG，无交互
- TotalRevenueCard 使用静态 SVG，无悬停提示
- EmailReports 使用 Recharts 但缺少 tooltip 和动画
- ProfileReportCard 使用静态 SVG

**改进方案：**
- 使用 Recharts 的 Tooltip 组件
- 添加响应式容器
- 添加入场动画
- 添加悬停高亮效果

### 4. 主体展示区域
**当前问题：**
- 卡片间距不够统一
- 缺少页面级的背景和内容区隔

**参考网站特征：**
- 内容区域有明确的 card 包裹
- 统一的 24px gutter
- 卡片圆角 0.5rem（8px）

---

## 实施步骤

### Step 1: 增强图表交互性 (优先级：高)
1. 重写 `StatCard.tsx` - 使用 Recharts 替换静态 SVG
2. 重写 `TotalRevenueCard.tsx` - 使用 Recharts BarChart + Tooltip
3. 重写 `ProfileReportCard.tsx` - 使用 Recharts AreaChart
4. 增强 `EmailReports.tsx` - 添加 Tooltip 和动画

### Step 2: 改进左侧导航栏
1. 调整折叠按钮位置到左侧边缘
2. 添加菜单项 hover 右侧箭头
3. 优化 active 状态样式
4. 添加底部用户卡片

### Step 3: 改进顶部状态栏
1. 优化搜索框样式
2. 添加通知下拉菜单
3. 优化用户头像下拉菜单

### Step 4: 优化主体布局
1. 统一卡片间距
2. 优化阴影效果
3. 添加内容区隔

---

## 验收标准

1. 图表有鼠标悬停提示框
2. 图表有入场动画
3. 导航栏 hover 效果流畅
4. 所有交互符合 Duralux 设计
